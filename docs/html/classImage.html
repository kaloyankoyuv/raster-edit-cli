◆ monochrome()

virtual void Image::monochrome ( )
pure virtual

Monochrome algorithm.

Turns grayscale or colored images to monochrome ones. Like grayscale but takes the average and compares it to the max value of the colors divided by two. If the average is greater, the pixels turns pure white, else pure black. Pure virtual so must be implemented by all derived classes.

Implemented in PBMImage, PGMImage, and PPMImage.

Negative algorithm.
